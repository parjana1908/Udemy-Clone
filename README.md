# Udemy-Clone
This project is a responsive static web page inspired by the Udemy online learning platform, built using HTML5 and CSS3. It replicates the core layout and visual structure of Udemy, focusing on clean UI design, course listing sections, and responsive components.
# Project Overview

The website showcases an online course platform interface with a navigation bar, category filters, promotional banner, recommended courses, popular courses, topic suggestions, and a structured footer. It is designed purely with frontend technologies and does not use JavaScript or backend services.

# Features

Responsive navigation bar with search input and icons

Course categories displayed as interactive pills

Promotional sales banner with call-to-action button

“Recommended for You” and “Most Popular” course sections

Reusable course cards with images, ratings, instructor names, and pricing

Topic recommendation section with hover effects

Clean footer layout with multiple links

Modern UI styling using Google Fonts and Font Awesome icons

# Technologies Used

HTML5 – Page structure and content

CSS3 – Styling, layout, flexbox, and hover effects

Google Fonts – Custom typography

Font Awesome – Icons for UI elements

# Purpose

This project is ideal for:

Frontend practice

UI/UX layout understanding

HTML & CSS portfolio showcase

Beginners learning responsive design using Flexbox
